Arrays

What is an array? Well an array is a block of consecutive memory locations that will hold values with the same data type. Each memory location within the array is called an 'element' and a specific element in an array is accessed using an index.

Declaring and initializing an array

When initializing an array, its size must be specified and after the array is created, its size cannot change.

ArrayList

The ArrayList class implements the List interface. One of the biggest differences between an array and ArrayList is the ability for an ArrayList to be resizable. Because of this, an ArrayList is best used when the the insertion or removing of elements is needed. Some of the most important methods from the ArrayList class are size(), set(), get(), and remove().

size()

This method returns an integer that represents the size of the ArrayList.

set(int index, E e)

This method replaces the element at the specified index in the ArrayList with the specified element.

get(int index)

Returns the element in the ArrayList at the specified index.

remove(int index)

Removes the element at the specified index from the ArrayList.

add(E e) and add(int index, E e)

add (E e) adds the element e to the end of the ArrayList. add(int index, E e) inserts the element e to the specified index (shifts all elements on and after the index to the right)
